# Zanchor — Portfolio

Minimal portfolio site skeleton, same stack and coding style as orderbot-tools:

- **Next.js 14** (App Router)
- **React 18**
- **Tailwind CSS** (dark theme, responsive)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

- `app/layout.js` — root layout, metadata, dark theme
- `app/page.js` — home page (hero + projects placeholder + footer)
- `app/globals.css` — Tailwind + shared utilities

Update copy, add pages under `app/`, and drop in components under `app/components/` as you grow the site.
